# AutoHotkeyUtils
This repository contains all of my AHK utility scripts.

- **opencmd** - Most useful one- Recreates Ctrl + Alt + T opening the terminal like in Linux. Opens in Administrator
- **10 min timer** - Makes a notification every 10 minutes. Useful for keeping track of the passage of time.
- **blenderbackdrag** - While in Blender and Unity, on a mouse with forward and back buttons, dragging the back button will orbit and the forward button will pan. Useful for mice with broken scroll clicking like mine.
- **CapsLockWithShift** - Disables the caps lock button unless pressed with CTRL + SHIFT + Caps Lock. Pressing Caps lock or shift while caps lock is on will disable it. Useful when you keep accidentally tapping it.
- **commit check** - Checks if committed memory usage is too high.
- **coolwindow** - Win+F minimizes all other apps and sets the current to a window slightly smaller than the screen it was originally on.
- **excel backspace** - Enables CTRL+Backspace in Excel, since it is not a native shortcut.
- **fullscreen** - Enables/Disables the title bar in windows that have one with Win+Left/Right click
- **holdw** - With W pressed, tapping Scroll Lock keeps W held until W or Scroll Lock is pressed again
- **mouseundoredo** - Binds Ctrl + Mouse Back/Forward to Undo/Redo
- **noctrlshiftwq** - Disables Ctrl + Shift + W/Q to prevent accidentally deleting all of your tabs
- **robloxrun** - Kills the lingering Roblox background task after close
- **Screen dimmer** - Dims screen on inactivity
